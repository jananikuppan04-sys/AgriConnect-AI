# Agricultural Seller Guidelines

## Product Listing Steps
Farmers and sellers must create an account and access their user panel to list agricultural goods. Standard listing creation is free of cost.

## Required Listing Details
To ensure buyer clarity and improve search results, each listing must contain:
* **Product Name**: Specific variety name (e.g., "Organic Alphonso Mangoes", "Basmati Rice Grade A").
* **Category**: Selection from Vegetables, Fruits, Grains, Spices, Dairy, or Equipment.
* **Quantity**: Accurate volume representation (e.g., "500 kg", "2 tonnes").
* **Price**: Unit price (e.g., "₹28 / kg").
* **Location/District**: The physical location where the produce is held for buyer pickup.
* **Available Date**: When the product is harvested and ready for transfer.
* **Image/Emoji**: High-quality visual representation representing product quality.

## Platform Listing Regulations
To maintain platform integrity, all sellers must follow these rules:
1. **Misleading Prices**: Do not set artificial base prices (e.g., ₹1) to generate clicks. The price must represent the actual transaction expectations.
2. **Duplicate Listings**: Do not publish identical listings. If you have different batches of the same product, update the quantity of the original listing.
3. **Prohibited Items**: Do not list non-agricultural goods, highly processed retail consumer packages, or restricted chemical compounds.
4. **Accuracy**: Promptly mark listings as sold or delete them when they are no longer available.
